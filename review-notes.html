ClipReplay TikTok Integration Review Notes

ClipReplay helps authorized creators upload one short-form video, localize captions for each destination, and publish the selected video to their connected TikTok account.

End-to-end flow

  1. The user signs in to TikTok through Login Kit and grants the requested posting scope.
  2. The user uploads an original short-form video inside ClipReplay.
  3. The user reviews or edits the TikTok caption and visibility settings.
  4. The user clicks the TikTok publish action.
  5. ClipReplay initializes a Content Posting API upload and transfers the video file to TikTok.
  6. The user can monitor status inside ClipReplay.

Scope usage

user.info.basic is used to show the connected TikTok account name in the channel connection screen.

video.upload and video.publish are used only after the user explicitly clicks publish for a selected video.
